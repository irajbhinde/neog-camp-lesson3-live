Useful VS Code Shortcuts That Would Save Your Time & Increase Productivity

1. Command Palette

Windows: CTRL + SHIFT + P
MAC: CMD+SHIFT+P

With the help of a command palette, you can access your commands according to your current context. You just need to type a keyword related to the command and find it, instead of navigating menus after menus.

2. Toggle Sidebar

Windows: CTRL + B
MAC: CMD+B

Sometimes while writing the code, you want to use the full width of the editor. Simple enough, just use this shortcut to hide or show your sidebar quickly. This will give you full width of the editor.

3. Multi-Select Cursor

Windows: CTRL + D
MAC: CMD + D

When you need to change the H3 tags to the H4 tags (all of them) in a file, what will you do? Usually, programmers go through each instance of H3 tag and change each tag. But, no! Thatâ€™s not how you do it. You can multi-select all the H3 tags and change them to H4 at once. For this Visual Studio Code shortcut, you have to select one H3 tag and use the shortcut. This will find every H3 tag and give you an active cursor for further action.

4. Copy Line

Windows: SHIFT+ALT+UP or SHIFT+ALT+DOWN
MAC: CMD + D

During software development, it is common for programmers to copy lines and paste them elsewhere in the project. After this, small details in the lines are changed. This is done so frequently that you can actually use a keyboard shortcut for quick copying/pasting.

5. Show Errors & Warnings

Windows: CTRL + SHIFT + M
MAC: CMD + SHIFT + M

Everybody likes bug-free code. This command shows the problems panel where you can see identified errors and warnings.

6. Multi Cursor Selection

Windows: CTRL + SHIFT + L
MAC: CMD + SHIFT + L

Add additional cursors to edit all occurrences of the current selection. Usually less error-prone than doing search and replace all.

Bonus Shortcuts To Try :

A) Change Multiple Lines Together

Windows: CTRL + ALT + Arrow Keys
MAC: CMD + ALT + Arrow Keys

B) JUMP TO LINE

Windows: CTRL + P + Type " : "
MAC: CMD + P + Type " : "

C) JUMP TO TAG

Windows: CTRL + P + Type " @ "
MAC: CMD + P + Type " @ "
